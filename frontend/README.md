# RoadAlert Frontend - Simple Login Page

## Quick Start

1. Open `index.html` in your browser
2. That's it! No installation needed.

## Files

- `index.html` - Main login page
- `style.css` - Styling
- `app.js` - Login logic

## Test the Login

- **Email**: any valid email format (e.g., `test@example.com`)
- **Password**: at least 6 characters

## Current Features

- Email validation
- Password length check (min 6 chars)
- Success/error messages
- Beautiful gradient UI

## Next Steps

When you're ready to add backend:
1. Create a Node.js server
2. Replace the TODO comment in `app.js` with actual API calls
3. Connect to PostgreSQL database

## Customization

Feel free to modify:
- Colors in `style.css` (search for `#667eea` and `#764ba2`)
- Form validation in `app.js`
- Add more fields (username, register, etc.)
